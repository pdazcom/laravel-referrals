# Laravel Referrals v2.4.0 Release Notes

Release date: 2026-03-21

## Highlights

- Added opt-in signup and first-purchase reward hooks for event-driven reward flows.
- Added duplicate reward prevention guard.
- Added opt-in self-referral prevention guard.
- Added `FixedRewardProgram` example with docs and tests for fixed credit payouts.
- Added order/subscription completion integration guide.
- Incorporated PR #48 review fixes.

## Versioning updates

- Package version bumped to `2.4.0` in `composer.json`.
- Laravel About command package version bumped to `2.4.0` in `ReferralsServiceProvider`.

## Upgrade notes

- No breaking changes.
- Existing integrations can update directly to `v2.4.0`.
