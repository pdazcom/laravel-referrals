# Laravel Referrals v2.3.0 Release Notes

Release date: 2026-03-21

## Highlights

- Added referral code as a first-class field on `ReferralLink`.
- Added configurable referral code generation strategy.
- Added uniqueness policy and stricter collision handling for manual and generated referral codes.
- Implemented code-based attribution flow.
- Added a detailed guide for referral sharing and referral entry flows.

## Versioning updates

- Package version bumped to `2.3.0` in `composer.json`.
- Laravel About command package version bumped to `2.3.0` in `ReferralsServiceProvider`.

## Upgrade notes

- No breaking changes.
- Existing integrations can update directly to `v2.3.0`.
